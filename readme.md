
# 📝 Lumina Log | 个人工作日志 + AI 智能总结系统

> **极致美观、沉浸式体验、AI 深度赋能的职场生产力工具。**

Lumina Log 是一款专为追求高效、专业的职场人士打造的个人工作日志管理系统。它不仅是一个记录工具，更是一个集成了 Google Gemini 3.0 顶级 AI 能力的智能汇报中心。

---

## 🌟 核心亮点

### 1. 极致视觉与交互
- **专业美学**：沉稳的蓝灰配色，大圆角卡片流设计，彻底摆脱“廉价工具感”。
- **沉浸式 UI**：专为桌面端优化，极简导航，让您的注意力始终集中在内容创作与数据洞察上。

### 2. AI 智能驱动 (Gemini 3.0 Core)
- **全自动总结**：基于任意自定义时间段，一键生成日报/周报/月报/年报。
- **内容重构**：AI 自动提取关键词、归纳工作成果、提炼待办事项、总结遇到的挑战与对策。
- **智能润色**：支持一键将口语化的记录转化为专业、得体的职场书面语。
- **自动分类**：无需手动打标签，AI 根据内容自动将其划分为研发、会议、管理等职能维度。

### 3. 全周期管理与导出
- **深度检索**：支持关键词精准搜索，快速回溯历史记录。
- **灵活导出**：支持将选定范围或单条记录导出为标准 Markdown (Word 样式) 或 CSV (Excel 样式)。
- **数据安全**：所有数据存储于本地浏览器，支持全量 JSON 备份与迁移。

## 🛠️ 技术架构

- **前端框架**: React 19 (Hooks, Functional Components)
- **样式方案**: Tailwind CSS (自定义设计系统)
- **AI 引擎**: @google/genai (Gemini 3.0 Flash/Pro)
- **持久化**: LocalStorage (本地离线优先)
- **构建/运行**: ESM-based browser runtime

## 🚀 快速开始

1. **配置环境**:
   系统依赖于 `process.env.API_KEY`。请确保您已拥有 Google AI Studio 的 API Key。

2. **核心逻辑**:
   - 点击左侧菜单进入 **“工作日志库”** 撰写日志。
   - 进入 **“智能汇报中心”** 选择时间范围，点击“开始生成”。
   - 在 **“系统设置”** 中自定义您的站点标题、Logo 及用户头像。
